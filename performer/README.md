# Aibo Performer

This folder contains preserved `Aibo Performer` materials associated
with the `ERS-110/111` era.

## Provenance

These materials were found via `Archive.org` after Sony had already
discontinued AIBO in `2006`.

That historical availability is useful for research and preservation,
but it should not be confused with a clear grant of redistribution
rights.

## Contents

- `Aibo Performer ERA-511 - ERA_111AP.iso`
- interface screenshots in `.png` format
- `checksum.txt` with the recorded `md5`

## Notes

- the preserved ISO currently records `md5 60869fe7f28ddfd91f6cd056d9f22b9d`
- the screenshots are included to document the software interface and help identify the correct disc image
- this folder is kept for historical and technical research around `ERS-110/111` tooling
